# 📌 Project Overview

This project analyzes global COVID-19 trends using confirmed cases, deaths, and recovery datasets.

The goal was to transform raw multi-sheet COVID-19 data into meaningful business and analytical insights through data cleaning, preprocessing, exploratory analysis, and visualization.

### Key Objectives

* Clean and preprocess raw COVID-19 datasets
* Convert wide-format data into analysis-ready format
* Merge multiple datasets into a unified structure
* Analyze confirmed cases, deaths, and recoveries
* Calculate recovery and fatality metrics
* Identify global trends and country-level patterns
* Create professional visualizations
* Generate actionable insights
